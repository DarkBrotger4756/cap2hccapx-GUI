GUI for cap2hccapx
Made By Operator4756


Программа создаёт 2 папки после первого запуска

В папке config находится файл DirPath.ini нужный для сохранения пути к 
вашему cap2hcapx.exe файлу

В папке Converted hccapx будут сконвертированные программой хэндшэйки


В программе есть 3 поля

Путь файла .cap - нажмите ... после чего откроется диалоговое окно в 
котором нужно выбрать ваш .cap файл

Директория - нажмите ... после чего откроется диалоговое окно в котором
нужно выбрать путь к вашему cap2hcapx.exe файлу

Название сети - Введите в это поле название взламываемой сети Wi-Fi


После того как вы нажмёте кнопку сконвертировать в папке Converted hccapx
будет ваш сконвертиованный в .hccapx формат файл
